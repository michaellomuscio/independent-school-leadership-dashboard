# Independent School Leadership Positions — Dashboard

Interactive dashboard for current independent-school leadership searches across Carney Sandoe, RG175, BlueGreen, Educational Directions, and other firms.

**[View the dashboard](https://michaellomuscio.github.io/independent-school-leadership-dashboard/)**

Features:
- Geographic map of position locations (US + international)
- Opportunities & Challenges theme analysis
- Reporting-structure breakdown, filterable by category
- Layered filters: search firm, category, state/region, theme
- Sortable position table with full metadata

The dashboard is regenerated from a private dataset and refreshed periodically.
